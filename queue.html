Queue

The epitome of a FIFO data structure

The Queue is the most classic example of a FIFO - First In First Out, data structure.
The Queue is a linear data structures that models real world queues, wherein elements are added to the back of the queue, and removed from the front.
This is why it is considered FIFO, the first element added is the first element removed.

An ADT of Queue commonly includes these four methods:

  • enqueue(element) – a method which adds a single element to the back of the queue, O(1)

  • dequeue() – a method which removes a single element from the front of the queue, O(1)

  • peek() - a method which returns the topmost element of the stack, without removal, O(1)

  • isEmpty() - a method which returns true if the queue is empty
